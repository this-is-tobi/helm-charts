Release "{{ .Release.Name }}" has been deployed in namespace "{{ .Release.Namespace }}".

================================================================================

Check deployment status:
  kubectl rollout status {{ .Values.servicename.deploymentType | lower }} {{ .Release.Name }}-servicename -n {{ .Release.Namespace }} -w

View all resources:
  kubectl get all -n {{ .Release.Namespace }} -l app.kubernetes.io/instance={{ .Release.Name }}

Follow logs in real-time:
  kubectl logs -n {{ .Release.Namespace }} -l app.kubernetes.io/name={{ .Release.Name }}-servicename -f

Port-forward to access the service locally:
  kubectl port-forward -n {{ .Release.Namespace }} svc/{{ .Release.Name }}-servicename 8080:{{ .Values.servicename.service.port }}
  Access at: http://localhost:8080
